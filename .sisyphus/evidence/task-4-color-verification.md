## Usage Patterns

### Primary Button (with Glow)

```css

  background: var(--color-primary);         /* #4f46e5 light, #818cf8 dark */
  color: var(--text-inverse);
  box-shadow: var(--glow-primary);          /* 0 0 24px indigo (dark only) */
```

- `--shadow-2`: Small (buttons)
