= Developer Documentation
:toc: left
:toclevels: 2

This directory contains internal development artifacts for the Boundary Framework project.
These documents are intended for contributors, maintainers, and team members — not end users.
They are plain AsciiDoc files readable directly in GitHub or any AsciiDoc viewer.

For user-facing documentation, see the link:../docs-site/[docs-site/] Hugo site.
