=== Babashka Developer Tooling

The `boundary-tools/` directory provides BB-based developer tools invoked via `bb.edn` tasks.
See link:../boundary-tools/AGENTS.md[boundary-tools/AGENTS.md] for the full reference.

Key tools:

* `bb doctor` — Validate config for common mistakes
* `bb setup` — Interactive config setup wizard (with optional AI mode)
* `bb scaffold integrate <module>` — Wire scaffolded modules into deps/tests/wiring
* `bb ai admin-entity <description>` — Generate admin entity EDN config via AI
* `bb deploy` — Deploy libraries to Clojars
* `bb create-admin` — Create first admin user
* `bb test:tools` — Run boundary-tools unit tests
