## Technical Details

### Conversion Process

```bash
# Step 1: Convert HTML to AsciiDoc
pandoc input.html -f html -t asciidoc -o output.adoc --wrap=none

# Step 2: Remove Antora navigation headers
awk 'BEGIN { found = 0 } /^==/ { found = 1 } found { print }' output.adoc

# Step 3: Verify and integrate
```

- **pandoc 3.8.2.1** - HTML to AsciiDoc conversion
