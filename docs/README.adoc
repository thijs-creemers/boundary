== Boundary Framework Documentation

Welcome to the Boundary Framework documentation. This documentation
covers architecture, guides, API references, and implementation details
for the Boundary Framework - a module-centric software framework built
on Clojure that implements the "Functional Core / Imperative Shell"
architectural paradigm.

=== Documentation Philosophy

This documentation follows these principles:

* *Clear and Concise* - Information is easy to find and understand
* *Example-Driven* - Real code examples for all patterns
* *Architecture-First* - Design principles guide implementation
* *Practical* - Focuses on real-world usage and common scenarios
* *Maintainable* - Simple structure without build complexity

=== Viewing Documentation

==== On GitHub

Simply browse the files directly - all documentation is readable in the
GitHub web interface.

==== Locally

You can view the documentation files directly in any text editor, or
serve them locally for a better browsing experience:

[source,bash]
----
# Smart serving with port allocation
npm run serve-docs

# Or use Python's built-in HTTP server
cd docs && python3 -m http.server 8080
----

Then visit http://localhost:8080 in your browser.

'''''

*No Build Step Required* - This documentation is designed to be browsed
directly without any build process. All files are standard Markdown
(.md) or AsciiDoc (.adoc) format.
