== Quick Commands

Here are the most common commands you'll use:

[source,{shell}]
----
# Run tests
clojure -M:test:db/h2

# Start REPL
clojure -M:repl-clj

# Start system (in REPL)
user=> (require '[integrant.repl :as ig-repl])
user=> (ig-repl/go)

# Use CLI
clojure -M:cli user list --tenant-id <UUID>

# Check for outdated dependencies
clojure -M:outdated
----

== Key Architectural Concepts

**Module-Centric Architecture:**  
Each domain module (`user`, `billing`, `workflow`) owns its complete functionality stack.

**Functional Core / Imperative Shell:**  
Strict separation between pure business logic (core) and side-effectful operations (shell).

**Ports and Adapters:**  
Hexagonal architecture pattern enabling dependency inversion and easy testing.

**Multi-Interface Support:**  
Consistent behavior across REST API, CLI, and Web interfaces.
