= API Reference

Complete API documentation and request/response examples for Boundary Framework. Each page covers one endpoint group: the HTTP contract, required fields, example payloads, and common error responses.

For implementation guidance — how to add a new endpoint or wire up validation — see the link:../guides/add-rest-endpoint[Add a REST API Endpoint] guide.
