== Scaffolding

[source,bash]
----
clojure -M -m boundary.scaffolder.shell.cli-entry generate \
  --module-name product \
  --entity Product \
  --field name:string:required \
  --field price:decimal:required
----

Generates: core, shell, ports, schema, tests, migration (12 files).
