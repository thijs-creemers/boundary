= 5-Minute Quickstart

Get your first Boundary application running in 5 minutes.

'''

== Prerequisites

* *Java 11+* (OpenJDK or Amazon Corretto)
* *Clojure CLI* (https://clojure.org/guides/install_clojure[install guide])
* *Babashka* (`brew install babashka` or https://babashka.org[install guide])

Verify your environment:

[source,bash]
----
java -version   # Should show 11 or higher
clojure -version
bb --version
----

'''

== Step 1: Clone & Setup

[source,bash]
----
git clone https://github.com/thijs-creemers/boundary.git
cd boundary
----

Download dependencies (cached after first run):

[source,bash]
----
clojure -P -M:repl-clj
----

'''

== Step 2: Configure Environment

Set required environment variables:

[source,bash]
----
export JWT_SECRET="dev-secret-32-chars-minimum-here"
export BND_ENV="development"
----

TIP: Add these to your shell profile (`.zshrc`, `.bashrc`) so they persist across sessions.

'''

== Step 3: Run Migrations

Apply the database migrations (uses H2 in-memory by default for development):

[source,bash]
----
clojure -M:migrate up
----

'''

== Step 4: Start the REPL

[source,bash]
----
clojure -M:repl-clj
----

The nREPL server starts on port 7888. Connect your editor, or use the REPL directly.

In the REPL, start the system:

[source,clojure]
----
(require '[integrant.repl :as ig-repl])
(ig-repl/go)
;; => :initiated
----

The HTTP server is now running at http://localhost:3000.

REPL lifecycle commands:

[source,clojure]
----
(ig-repl/reset)  ;; Reload changed namespaces and restart
(ig-repl/halt)   ;; Stop the system
(ig-repl/go)     ;; Start the system
----

'''

== Step 5: Run the Tests

Open a second terminal and verify everything works:

[source,bash]
----
clojure -M:test:db/h2
----

All tests should pass. To run only one library:

[source,bash]
----
clojure -M:test:db/h2 :user
----

'''

== Step 6: Scaffold Your First Module

Use the interactive scaffolder to generate a new module:

[source,bash]
----
bb scaffold
----

The wizard will prompt you for:

. Module name (e.g., `tasks`)
. Entity name (e.g., `Task`)
. Fields (e.g., `title:string`, `completed:boolean`)

The scaffolder generates a complete module under `libs/tasks/`:

[source,text]
----
libs/tasks/src/boundary/tasks/
├── core/
│   ├── tasks.clj       # Pure business logic
│   └── validation.clj  # Validation rules
├── shell/
│   ├── http.clj        # HTTP handlers & routes
│   ├── persistence.clj # Database adapter
│   └── service.clj     # Shell orchestration
├── ports.clj           # Protocol definitions
└── schema.clj          # Malli schemas
----

After scaffolding, run the generated tests:

[source,bash]
----
clojure -M:test:db/h2 :tasks
----

'''

== Common Issues

=== REPL won't start

[source,bash]
----
# Clear dependency cache and retry
rm -rf .cpcache/
clojure -M:repl-clj
----

=== Tests failing

Ensure the JWT_SECRET environment variable is set:

[source,bash]
----
JWT_SECRET="dev-secret-32-chars-minimum" clojure -M:test:db/h2
----

=== Port 3000 already in use

[source,bash]
----
lsof -ti:3000 | xargs kill -9
----
