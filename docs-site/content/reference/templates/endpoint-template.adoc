[[_request]]
===== link:#_request[]Request

*Method:* `++[++GET++|++POST++|++PUT++|++DELETE++]++` *Path:* `/api/v1/++[++resource++][++/++{++id}++]++` *Content-Type:* `application/json`

*Path Parameters:*

[width="100%",cols="20%,20%,60%",options="header",]
|===
|Parameter |Type |Description
|id |UUID |++[++Description of path parameter++]++
|===

*Query Parameters:*

[width="99%",cols="18%,16%,16%,50%",options="header",]
|===
|Parameter |Type |Required |Description
|limit |integer |No |Maximum number of results (default: 20, max: 100)
|offset |integer |No |Number of results to skip (default: 0)
|===

*Request Body Schema:*

[source,highlightjs,highlight]
----
{:example/field [:string {:min 1 :max 100}]
 :example/id    :uuid
 :example/flag  :boolean}
----

*Example Request:*

[source,highlightjs,highlight]
----
{
  "exampleField": "value",
  "exampleId": "123e4567-e89b-12d3-a456-426614174000",
  "exampleFlag": true
}
----
