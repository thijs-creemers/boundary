## About

Boundary is a production-ready framework featuring:

- **Functional Core / Imperative Shell** architecture
- **10 independently publishable libraries**
- **Zero-config database** (SQLite for development)
- **Built-in observability** (logging, metrics, error reporting)
- **Authentication & MFA** out of the box
- **Auto-CRUD admin interface**
- **Module scaffolding** for rapid development
