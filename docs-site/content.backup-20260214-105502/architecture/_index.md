Comprehensive documentation of the Boundary Framework's architecture, design patterns, and system components.

## Core Concepts

- **Functional Core / Imperative Shell** - The foundational architectural pattern
- **Ports and Adapters** - Dependency injection and abstraction
- **Layer Separation** - Clear boundaries between business logic and infrastructure
