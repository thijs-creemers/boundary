# Examples

Complete, working example applications demonstrating Boundary Framework patterns and features.

## Example Applications

Content will be migrated from the boundary-docs repository, including:

### Todo API
A simple REST API demonstrating:
- CRUD operations
- Validation
- Database persistence
- Error handling

### Blog Application
A complete blog platform showing:
- User authentication
- Entity relationships
- File uploads
- Admin interface

### E-Commerce API
An e-commerce backend featuring:
- Product catalog
- Order processing
- Payment integration
- Inventory management

### Notification Service
A background job service demonstrating:
- Async job processing
- Email notifications
- Scheduled tasks
- Error recovery

## Learning from Examples

Each example includes:
- **Complete source code** - Fully functional implementation
- **Setup instructions** - How to run locally
- **Key concepts** - What patterns are demonstrated
- **Extension ideas** - How to build on the example
